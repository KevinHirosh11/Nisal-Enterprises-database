# Nisal Enterprises – Stock Management System

## 📌 Overview

**Nisal Enterprises Stock Management System** is a web-based software solution developed to efficiently manage inventory, track stock levels, and handle product records. This system helps businesses monitor stock movement, reduce manual work, and improve accuracy.

The application is built using:

* **Backend:** Python (Flask API)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** Connected via Flask (SQL Database such as MySQL)

---

## 🚀 Features

* 📦 Add new products
* 📝 Update product details
* ❌ Delete products
* 📊 View available stock
* 🔍 Search products
* 📈 Track stock quantity
* 🔐 Backend API integration using Flask
* 💻 User-friendly interface

---

## 🏗️ System Architecture

Frontend (HTML, CSS, JavaScript)
⬇
Flask API (Python Backend)
⬇
Database (MySQL)
